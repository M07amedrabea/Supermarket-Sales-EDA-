# Supermarket Sales Data Analysis

## Overview

This project analyzes supermarket sales data using Python. It provides insights into various aspects such as payment methods, customer demographics, and product line distributions.

## Features

- Reads sales data from a CSV file
- Performs exploratory data analysis (EDA)
- Generates visualizations for different features
- Identifies trends and distributions in the data

## Installation

Ensure you have Python installed. Install the required libraries using:

```sh
pip install pandas numpy matplotlib seaborn
```

## Usage

1. Place the dataset `supermarket_sales - Sheet1.csv` in the working directory.
2. Run the script to generate insights and visualizations:

```sh
python analysis.py
```

## Data Insights

- Distribution of payment methods
- Customer demographics (gender, branch, city)
- Product line popularity
- Trends in total sales and customer ratings

## Visualization

- Bar charts for categorical distributions
- Line plots for numerical trends
- Pie charts for gender distribution
